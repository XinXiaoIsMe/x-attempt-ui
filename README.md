# x-attempt-ui

## 安装
```bash
# pnpm
pnpm i x-attempt-ui

# npm
npm i x-attempt-ui

# yarn
yarn add x-attempt-ui
```

## 使用
```ts
// main.ts
import { createApp } from 'vue'
import XAttemptUI from 'x-attempt-ui'
import 'x-attempt-ui/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(XAttemptUI)
app.mount('#app')
```
